File Upload Instructions

Please follow the instructions below when uploading an image or PDF related to a victim's case. This will help us organize and identify the files accurately.

File Naming Convention

Before uploading a file, rename it using the following format:

Format: VictimName_IncidentType_Date.FileExtension

Examples:

  • For a Death Case:
    AbuSaeed_Death_2024-08-15.jpg
    (Victim's Name: Abu Saeed, Incident Type: Death, Date: August 16, 2024, Image File)
  • For a Missing Case:
    NahidIslam_Missing_2024-07-23.pdf
    (Victim's Name: Nahid Islam, Incident Type: Missing, Date: July 20, 2024, PDF Report)
  • For an Injured Case:
    RabiulIslam_Injured_2024-09-01.pdf
    (Victim's Name:Rabiul Islam, Incident Type: Injured, Date: September 1, 2024, PDF Medical Report)

File Upload Guidelines:

  • Image files: Maximum size 1MB, allowed formats: .jpg, .jpeg, .png
  • PDF files: Maximum size 5MB, allowed format: .pdf
